# FundExecs Technologies — Business Model Canvas

*Confidential · July 2026*

---

## 1. Customer segments

**Beachhead:**
- Emerging managers & first-time GPs (PE, real estate, credit) raising Funds I–III
- Independent sponsors and deal-by-deal syndicators
- Single- and multi-family offices running direct investments

**Expansion:**
- Established lower-middle-market GPs ($100M–$1B AUM)
- Placement/advisory firms and fund-service providers
- Eventually: LPs/allocators (portal is already in-product) and lenders

## 2. Value propositions

- **Time:** compress the ~80% of operator time spent moving information — sourcing sweeps, IC memos, diligence summaries, LP updates — from hours to minutes.
- **Headcount leverage:** an emerging manager gets the output of an analyst, an associate, an IR function, and a fund-admin prep desk for a software subscription.
- **Control:** approval-gated by default; nothing reaches a counterparty without sign-off — the only AI posture a fiduciary can adopt.
- **Compounding memory:** three native graphs + org Brains mean the platform gets smarter with every workflow; switching cost grows with every deliverable.
- **One system of record:** replaces 30+ point solutions and their integration tax.

## 3. Channels

- Founder-led sales into the founder's operating network (PE/family-office advisory relationships)
- **Capital-formation salons** (The $100M Room format) — rooms full of exactly the ICP, run by the Curator/Capital Raiser playbooks the product itself encodes
- Product-led: self-serve signup, demo seed + guided tour, credit packs as low-friction entry
- Referral program (in-product: referral credits and gifts are live)
- Content/SEO motion (the SEO Disruptor playbook applied to ourselves)
- Later: fund-services partnerships (fund admins, placement agents) as resellers

## 4. Customer relationships

- Design-partner intimacy in year one: weekly working sessions, roadmap co-creation
- Self-serve + in-product guided tour for the long tail
- Dedicated success for Scale-tier firms (SLA, onboarding, governance support)
- The product itself maintains the relationship: agents surface value proactively (digests, radar, proposals)

## 5. Revenue streams

| Stream | Mechanics | Status |
|---|---|---|
| **Subscription plans** | Starter / Pro / Scale per-org tiers with seat limits (1 / 10 / unlimited) — live in-product at early-access pricing ($5 / $30 / $100 per month; annual = 10× monthly) | Live |
| **Metered credits** | Agent execution burns credits; credit packs ($5–$90) top up beyond plan allotments — usage revenue scales with work delegated | Live |
| **Institutional repricing** | As deliverable value is proven, tiers reprice toward the value of work replaced (target: $300–$2,500/org/mo blended; see document 08) | Planned Y1–Y2 |
| **Marketplace take-rate** | Listings + matching between orgs (deal shares, services, co-invest) | Schema live; logic planned |
| **Data / API grants** | Scoped programmatic access for partners and LPs | Plumbing live |
| **Trust layer** | Reputation-based entitlements, attestations, verified track record — premium governance features | Specified |

## 9. Cost structure

- **Engineering payroll** — dominant cost (see documents 08–09)

- **Infrastructure** — Supabase/Vercel/AWS; modest at current scale
- **GTM** — founder-led + salons (low CAC by design); success headcount from Y2
- **Legal/compliance** — entity/IP hygiene, securities counsel for trust layer, SOC 2

## Unit economics logic (summary)

Credits create a direct line from *work delegated* → *revenue*, so net revenue retention is driven by agent adoption rather than seat growth alone. Gross margin target 75–85% (SaaS-typical minus AI compute; compute per deliverable falls as model prices fall and routing improves). CAC stays low while the wedge is founder-network + salon + product-led; the marketplace and trust layers add high-margin expansion revenue on the same base.
